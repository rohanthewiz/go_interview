# Go Interview

### Database
The SQLite DB is cass_notes.sqlite

The notes table columns are:
- id integer
- guid varchar(40)
- title varchar(128)
- description varchar(255)
- body text
- tag varchar(128)
- user varchar(255) - not used
- creator varchar(255) - not used
- shared_by varchar(255) - not used
- public bool
- created_at datetime
- updated_at datetime
